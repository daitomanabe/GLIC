// About

What is GLIC?

An experimental image codec designed to create controlled glitch effects through intentional manipulation of compression algorithms.

Artistic Compression

GLIC reinterprets traditional image compression techniques as creative tools. By exposing and manipulating prediction errors, quantization, and wavelet transforms, it generates unique visual artifacts.

Controlled Chaos

Unlike random data corruption, GLIC provides precise control over glitch parameters. Adjust prediction methods, color spaces, and encoding techniques to achieve consistent, reproducible results.

Multiple Implementations

Available as both a Processing-based GUI application for interactive exploration and a C++ command-line tool for batch processing and integration into automated workflows.

// Usage

Quick Start

Get started with the C++ command-line interface

Build from source 
# Clone and build
cd implementation/glic_cpp
mkdir build && cd build
cmake ..
cmake --build .
Basic encode/decode 
# Encode image to GLIC format
./glic encode photo.png glitched.glic

# Decode back to image
./glic decode glitched.glic result.png
Creative examples 
# Spiral prediction with high quantization
./glic encode photo.png out.glic --prediction SPIRAL --quantization 180

# Wave effect in YUV color space
./glic encode photo.png out.glic --colorspace YUV --prediction WAVE

# Apply post-effects on decode
./glic decode out.glic result.png --effect scanline --effect chromatic

# Combine multiple effects
./glic decode out.glic result.png --effect posterize --effect-levels 4 --effect glitch
